# CURSOR_TECHMAP

Google Apps Script для автоматизации создания технологических карт в Google Sheets.

## Что делает решение

- добавляет меню `Техкарты`
- хранит библиотеку шаблонов в компактном каталоге на скрытых листах `_TC_LIBRARY` / `_TC_STORE`
- открывает боковую панель с вкладками **Техкарты**, **Техоперации**, **Материалы**
- вставляет шаблон в текущую активную ячейку с переносом форматирования, объединений, размеров, формул и изображений
- позволяет технологу сохранить любой выделенный диапазон как новый шаблон без правки кода

## Структура проекта

| Файл | Назначение |
|------|-----------|
| `Code.gs` | Константа `TECHMAP_APP`, кастомные функции листа, публичные точки входа |
| `TemplateStore.gs` | Внутренняя реализация: каталог, хранилище, копирование, изображения |
| `Materials.gs` | Загрузка, кэширование и поиск базы материалов |
| `OperationDatabase.gs` | Загрузка, кэширование и поиск базы техопераций |
| `WorkspaceSidebar.html` | Единая рабочая панель (три вкладки) |
| `SaveTemplateDialog.html` | Диалог сохранения / обновления шаблона |
| `appsscript.json` | Манифест проекта (scopes, timezone) |

## Как развернуть

1. Создайте Google Spreadsheet.
2. Откройте `Extensions → Apps Script`.
3. Создайте файлы `Code.gs`, `TemplateStore.gs`, `Materials.gs`, `OperationDatabase.gs`, `WorkspaceSidebar.html`, `SaveTemplateDialog.html` и вставьте содержимое из репозитория.
4. Сохраните проект и обновите таблицу.
5. В меню `Техкарты` откройте **Открыть рабочую панель** — при первом запуске Google запросит разрешения.

## Как пользоваться

### Вставка шаблона

1. Выберите ячейку на рабочем листе.
2. Откройте `Техкарты → Открыть рабочую панель`.
3. На вкладке **Техкарты** найдите нужный шаблон и нажмите **Вставить**.

### Сохранение шаблона

1. Нарисуйте блок на рабочем листе.
2. Выделите диапазон.
3. Откройте `Техкарты → Сохранить выделение как шаблон`.
4. Укажите название, категорию, описание — готово.
